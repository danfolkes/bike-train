Bike Train
==========

Use Github GUI to checkout somewhere on your computer.

By default the code is setup with Richmond, VA Bike Train. So you can view the database readme for adding your routes.

Then, you should just be able to upload it to a web server that has:
 * PHP
 * SQLite3 For PHP
   - http://php.net/manual/en/book.sqlite3.php
   - sudo apt-get install php5-sqlite
 * The db folder must be writable and possibly executable
